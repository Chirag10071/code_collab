Getting Started

Welcome to the CodeCollab developer documentation. This guide will help you integrate with our collaborative coding platform and use our API to build amazing developer tools.

Prerequisites

  • CodeCollab developer account
  • API key (available in your account settings)
  • Basic understanding of REST APIs

Installation

For JavaScript projects, install our official SDK:

npm install codecollab-sdk

Or include directly in your HTML:

<script src="https://cdn.codecollab.io/sdk/v2.4.1/codecollab.min.js"></script>

API Reference

All API endpoints require authentication via your API key. Include it in the X-API-Key header with each request.

GET /api/v1/projects

Retrieve a list of all projects you have access to.

POST /api/v1/projects

Create a new project.

Request Body

Parameter Type Required Description
name string Yes Project name (3-50 characters)
description string No Project description (max 200 chars)
template string No Template ID to base project on
visibility string No "private" or "public" (default: private)
GET /api/v1/projects/{id}

Get details about a specific project.

PUT /api/v1/projects/{id}

Update project details.

DELETE /api/v1/projects/{id}

Delete a project.

Authentication

All API requests must include your API key in the headers:

X-API-Key: your_api_key_here

For user authentication (OAuth2 flow):

POST /oauth2/token
Content-Type: application/x-www-form-urlencoded

grant_type=authorization_code
&code=AUTHORIZATION_CODE
&redirect_uri=REDIRECT_URI
&client_id=CLIENT_ID
&client_secret=CLIENT_SECRET

Working with Projects

Creating a Project

Here's how to create a new project using our JavaScript SDK:

const CodeCollab = require('codecollab-sdk');
const client = new CodeCollab('YOUR_API_KEY');

client.projects.create({
    name: 'My Awesome Project',
    description: 'A collaborative coding project',
    template: 'nodejs-express',
    visibility: 'private'
})
.then(project => {
    console.log('Project created:', project);
})
.catch(error => {
    console.error('Error:', error);
});

Real-time Collaboration

Connect to real-time editing sessions:

const socket = client.realtime.connect(projectId);

socket.on('connect', () => {
    console.log('Connected to real-time session');
});

socket.on('code-update', (update) => {
    // Handle incoming code changes
    console.log('Update received:', update);
});

// Send an update
socket.emit('code-update', {
    file: 'main.js',
    changes: [{ line: 10, text: 'console.log("Hello World");' }]
});

Code Examples

Python Example

import requests

api_key = "your_api_key"
headers = {"X-API-Key": api_key}

# Get all projects
response = requests.get("https://api.codecollab.io/api/v1/projects", headers=headers)
projects = response.json()

print("Your projects:")
for project in projects:
    print(f"- {project['name']} (ID: {project['id']})")

JavaScript Fetch Example

const apiKey = 'your_api_key';
const headers = { 'X-API-Key': apiKey };

// Create a new project
fetch('https://api.codecollab.io/api/v1/projects', {
    method: 'POST',
    headers: {
        ...headers,
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        name: 'New Project',
        description: 'Created via API'
    })
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

Frequently Asked Questions

How do I get an API key?

API keys are available in your account settings under "Developer Options". Each user can generate up to 5 API keys.

What's the rate limit?

Free accounts have a limit of 100 requests per minute. Paid plans have higher limits.

How do real-time updates work?

We use WebSockets for real-time collaboration. When connected, you'll receive updates whenever other collaborators make changes.

Can I use CodeCollab for private projects?

Yes, by default all projects are private. You can invite specific collaborators or make a project public if desired.
